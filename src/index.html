Создать граф

Задайте имена всех вершин — они отображаются внутри кругов вместо номеров.

Первая строка: n коротких имён (до 3 символов) через пробел. Далее m строк «f l v» (начало, конец, вес); вершины 1…n. Петля: одинаковые f и l. Для Дейкстры, TSP и др. кратные дуги (одинаковые f,l) в матрице сводятся к минимальному v; на экране рёбра по-прежнему отдельные.

Введите матрицу смежности

Пример квадратной матрицы:
0 1.5 0 2.3 1
1.5 0 1.0 0 8
0 1.0 0 1.7 7
2.3 0 1.7 0 4
0 0 0 0 0

Транспортная задача (минимум затрат)

Строка 1: m n — число пунктов отправления и назначения.
Далее m строк по n чисел — матрица тарифов cij.
Затем одна строка: m запасов ai, и одна строка: n спросов bj.
Строки, начинающиеся с #, игнорируются. При Σa ≠ Σb добавляется фиктивный пункт с нулевыми тарифами.
